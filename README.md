# iguanes.club

Static site for Les Iguanes.

## GitHub Pages

GitHub Pages can serve this repository directly from the repository root because
the site entry point is `index.html`.

Main files:

- `index.html`
- `iguanes-v3.css`
- `iguanes-v3.js`
- `assets-web/`
- `assets-mobile/`
- `assets-tablet/`
- `scripts/build-responsive-media.mjs`
- `*.zip` source archives at the repository root

Local verification:

```bash
bash tests/github-pages-smoke.sh
```
