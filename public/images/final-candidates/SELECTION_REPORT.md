# Selection Report

## Scope

This first pass focuses on fixed images only.

- Goal now: `Accueil`, `Histoire`, `Galerie`
- Deferred: `Communication / réseaux sociaux`, `Effectif`, video integration
- Principle: short, high-quality, story-driven selection over exhaustive archive

## Counts

- Images reviewed in the main catalog: `432`
- Detail:
  - `235` still images from `media/`
  - `197` prepared still images from `assets-web/`
- Not counted as new sources:
  - `assets-mobile/` and `assets-tablet/` because they are derivative exports
  - `public/images/selected-review/` because it is a prior shortlist, not source media
- Unique images retained and copied to this folder: `21`

## Editorial Method

Selection criteria used for every retained image:

- visual quality
- diversity of scenes
- representation of the club
- emotion
- strong moments: team, match, finale, celebration, community

Images were removed when they were:

- near-duplicates from the same sequence
- burst shots with no added narrative value
- softer or weaker alternatives to a better frame
- badly framed
- too static without strong storytelling value
- less flattering than other available options
